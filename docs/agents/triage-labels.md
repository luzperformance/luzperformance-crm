# Triage labels

The canonical triage labels in this repo are:

| Role | Label |
| --- | --- |
| needs-triage | `needs-triage` |
| needs-info | `needs-info` |
| ready-for-agent | `ready-for-agent` |
| ready-for-human | `ready-for-human` |
| wontfix | `wontfix` |

Use these exact label strings when triaging GitHub issues in this repo.
